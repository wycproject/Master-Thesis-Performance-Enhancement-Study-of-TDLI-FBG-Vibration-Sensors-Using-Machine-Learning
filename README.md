# Master-Thesis-Performance-Enhancement-Study-of-TDLI-FBG-Vibration-Sensors-Using-Machine-Learning

# 機器學習結合可調干涉儀與光纖光柵震動感測器之效能提升研究
這是我在國立臺北科技大學人工智慧科技碩士班的學位論文核心程式碼。

### 專案亮點
- **數據處理**：處理光纖感測器的原始訊號，進行去噪與特徵提取。
- **模型架構**：使用 PyTorch 實作 DNN 模型。
- **優化成果**：透過訓練優化，顯著提升感測系統的靈敏度。

### 技術棧
- Python, PyTorch, NumPy, Matplotlib
